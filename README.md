# BruceButBetter — prebuilt firmware bins

Asset host for the [BruceButBetter](https://github.com/Yoursel71/BruceButBetter) web flasher.
Each `Bruce-<board>.bin` is a merged image (flash at offset `0x0`). Served same-origin via GitHub Pages so
the browser flasher can fetch any board.

Not for cloning — see the main repo. Built from BruceButBetter `v1.0.0`.
